# CSS 🎨

Welcome to the **CSS** repository! This collection serves as a hands-on workspace for practicing Cascading Style Sheets, focusing on advanced selectors, layout techniques, and interactive UI mini-projects.

## 📖 About This Repository
This repository contains various HTML and CSS files demonstrating frontend styling techniques. It ranges from fundamental CSS concepts—like combinatorial targeting and external styling—to more complex, fully-styled components and application interfaces.

## 🗂️ Concepts & Projects Covered

### Core CSS Concepts
* **Selectors & Targeting:** Advanced targeting using pseudo-classes, pseudo-elements, and combinators (`selectors.html`, `pseudo-class Selector.html`, `Pseudo-element Selector.html`, `SelectorComibator.html`).
* **Styling Methods:** Implementing and linking external stylesheets (`externalmethod.html`, `externalmethod.css`).
* **Visual Effects:** Working with colors, backgrounds, and gradients (`garient.html`).

### UI Components & Mini-Projects
* **Application Interfaces:** Recreating functional-looking UIs like an ATM machine (`atm_machine.html`), a calculator (`calculator.html`, `calculator2.html`), and a PhonePe app interface (`phonepe_app.html`).
* **Web Components:** Building styling components like navigation bars (`navbar.html`, `navbar.css`) and structured data tables (`foodtable.html`).
* **Creative Layouts:** Designing a clean, structured resume (`resume.html`) and exploring visual layouts like a solar system representation (`solar system.html`).
